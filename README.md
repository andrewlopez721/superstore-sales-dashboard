# superstore-sales-dashboard
Analyzed 4+ years of retail sales data for a fictional superstore to uncover trends, identify underperforming areas, and provide actionable recommendations to increase profitability. Used Tableu public to visualize and clean data sets.
## Dataset
- Source: Microsoft excel Superstore Sales Dataset
- ~10,000+ rows, 4 years of orders
- Key columns: Order Date, Sales, Profit, Category, Region, Segment, etc.
- ## Key Insights
- Technology category generated the highest sales but Furniture had the lowest profit margin (X%).
- manufacturers out west were more profitable
